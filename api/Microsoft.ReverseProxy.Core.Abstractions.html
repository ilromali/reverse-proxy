Classes

Backend

A backend is a group of equivalent endpoints and associated policies. A route maps requests to a backend, and Reverse Proxy handles that request by proxying to any endpoint within the matching backend, honoring load balancing and partitioning policies when applicable.

BackendPartitioningOptions

Backend partitioning configures which endpoints in a backend are eligible to process each request.
