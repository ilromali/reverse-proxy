public class RecurrentActivityManager : IRecurrentActivityManager
Remarks

This class is optimized for very simple background tasks. Different activities can be executed at the same time, but only one instance of a given activity is running at any point. If an activity is still running when its next scheduling comes up, the next scheduling is skipped -- skipping will continue until the previous one completes.
