Remarks

ISO 8601 (also documented in RFC3339 to some degree) is the duration standard commonly used in Azure for encoding durations. For example, the azure VMSS autoscale settings (https://docs.microsoft.com/en-us/azure/azure-monitor/platform/autoscale-understanding-settings) use this format for specifying 'timeWindow' and 'timeGrain' parameters.
